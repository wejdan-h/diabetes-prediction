# Diabetes Prediction

## Overview

This project applies machine learning techniques to predict diabetes using clinical patient data. The project includes exploratory data analysis (EDA), data preprocessing, feature scaling, model training, and performance evaluation using Logistic Regression.

## Exploratory Data Analysis (EDA)

The dataset was explored using:

- Data preview
- Descriptive statistics
- Missing value checking
- Duplicate detection
- Line Plot
- Histogram
- Correlation Heatmap
- Scatter Plot

## Machine Learning Model

- Logistic Regression

## Data Preprocessing

- Missing value checking
- Duplicate removal
- Feature standardization using StandardScaler
- Train/Test split

## Evaluation

The model was evaluated using:

- Accuracy Score
- Confusion Matrix
- Classification Report

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Files

- Diabetes_Prediction.ipynb
- diabetes.csv

## Purpose

This project demonstrates the complete workflow of a machine learning classification task, including exploratory data analysis, data preprocessing, model training, and performance evaluation.
